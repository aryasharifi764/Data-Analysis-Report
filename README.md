# Data-Analysis-Report
Assignment using python in order to analyse data from a large dataset from Lending Club to analyse the determinants of interest rates in the consumer credit market.

Repository contents:
Data_Report.ipynb - Google Colab notebook with all data preparation, exploratory analysis, visualisations, and regression models (Parts A–E of the assignment).
loans_dataset.csv – Original Lending Club dataset provided by the instructor.

How to reproduce the analysis:
1. Download this repository
2. Open loans_analysis.ipynb in Google Colab
3. Ensure loans_dataset.csv is available in Colab
4. Run all cells in order from top to bottom

Software and Packages:
1. pandas, numpy for data handling.
2. matplotlib, seaborn for visualisations.
3. statsmodels for estimation of the linear regression models.
